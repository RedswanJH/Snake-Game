# Java Snake Game - Assignment 1

This is a classic Snake Game developed in Java for the course **159.261 Games Programming**. It features both Single Player and Two-Player (PVP) modes.

You can watch the full game demonstration on Bilibili:
[Click here to watch my Video] ## ✨ Features
- **PVP Mode**: Two players can play on the same keyboard (Green vs Blue).
- **Life System**: Each player starts with 4 lives (heart icons).
- **Poison Apples**: Eating a "poison apple" reduces 1 life.
- **Dynamic Growth**: Snake grows after eating apples (Max length: 20).
- **Game Menu**: Professional UI with Start, Help, and Exit options.

## ⌨️ Controls
| Action | Player 1 (Green) | Player 2 (Blue) |
| :--- | :--- | :--- |
| **Move** | Arrow Keys | W, A, S, D |
| **Menu** | Space (Restart) | - |

## 🛠️ How to Run
1. Download the repository as a ZIP or clone it.
2. Ensure you have **Java JDK** installed.
3. Keep the `resources` folder in the same directory as the `.java` files.
4. Compile: `javac SnakeGame.java`
5. Run: `java SnakeGame`

## 📁 File Structure
- `src/`: Source code (`SnakeGame.java`, `GameEngine.java`)
- `resources/`: Images (apples, heads, hearts, etc.)
- `README.md`: Project documentation
